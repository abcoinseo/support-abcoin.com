Overview

Our API provides access to various features of the AB Coin ecosystem, such as token transfer, account balances, and transaction history. You can easily integrate these features into your application with simple HTTP requests.

Authentication

To interact with the AB Coin API, you must first authenticate using an API key. You can obtain your API key by registering on our platform.

Get API Key

To get your API key, sign up on our platform and navigate to the 'API Settings' page.

GET /api/v1/authenticate

API Endpoints

GET /api/v1/coin/balance

Retrieve the balance of a specific wallet address.

GET /api/v1/coin/balance?address={wallet_address}

Response Example:

{
    "address": "AB3c17HWUjVr7ueKYW8_isGzyfLnQT3Umg2JYdW-6tBWq6t",
    "balance": "1000.50"
}

POST /api/v1/coin/transfer

Transfer tokens from one address to another.

POST /api/v1/coin/transfer
{
    "from": "{sender_address}",
    "to": "{receiver_address}",
    "amount": "100"
}

Response Example:

{
    "status": "success",
    "transaction_id": "abcd1234"
}
